# 当前用户流程
## Current User Flow

---

## 📍 流程概览

```
首页 → 规划页面（5个标签页）→ 查看结果
```

---

## 🚀 详细步骤

### Step 1: 首页（HomePage）
- **操作**：点击 "Start Planning" 按钮
- **结果**：进入规划页面

---

### Step 2: 规划页面（PlannerPage）

页面分为**左侧标签页**和**右侧可视化区域**

#### 左侧：5个标签页（按顺序使用）

**① Configure（配置房间）**
- 选择房间类型（客厅/卧室/餐厅/书房）
- 输入房间尺寸（长、宽、高）
- 选择单位（米/英尺）
- 点击完成

**② Preferences（偏好设置）**
- 输入预算（可选）
- 选择家具类别（多选）
- 选择产品系列（可选）
- 点击"获取推荐" → AI自动推荐家具

**③ Image（图片处理）**
- **上传房间图片**
- **AI自动检测**：
  - 如果检测到家具 → 显示检测结果，可选择替换
  - 如果是空房间 → 可手动搜索并放置家具
- **替换/放置家具** → 渲染到图片中

**④ Chat（AI聊天）**
- 与AI对话，询问建议
- 支持中英文

**⑤ Cart（购物车）**
- 查看已选家具
- 调整数量
- 结算（待实现）

#### 右侧：可视化区域
- **2D/3D视图切换**
- **家具列表**：显示已放置的家具
- **操作**：添加到购物车、移除家具

---

## 💡 关键特点

1. **标签页式导航**：用户可以自由切换标签，不强制顺序
2. **实时可视化**：右侧实时显示房间设计和家具
3. **AI自动推荐**：在Preferences标签点击"获取推荐"后自动生成
4. **图片处理**：支持上传图片、AI检测、替换/放置家具

---

## 🔄 典型使用路径

**路径A：传统规划**
```
Configure → Preferences → 获取推荐 → 查看右侧结果 → Chat咨询 → Cart
```

**路径B：图片驱动**
```
Image → 上传图片 → AI检测 → 替换家具 → 查看结果 → Cart
```

**路径C：混合使用**
```
Configure → Preferences → Image → Chat → Cart
```

---

## 📝 注意事项

- 标签页可以**自由切换**，没有强制顺序
- **Preferences** 中的"获取推荐"会触发AI推荐
- **Image** 标签支持两种模式：替换现有家具 / 空房间放置
- 所有操作结果都在**右侧可视化区域**实时显示
